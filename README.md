# Защищенное веб-приложение на Spring Security

## Описание
Веб-приложение демонстрирует защиту страниц с помощью Spring Security. 
Пользователь может просматривать публичную домашнюю страницу, но для доступа к странице приветствия требуется аутентификация через форму входа. 
Приложение использует встроенное хранилище пользователей с одним тестовым аккаунтом.

## Стек технологий
- **Backend:** Java 17, Spring Boot, Spring Security, Spring MVC
- **Frontend:** Thymeleaf, HTML
- **DB:** In-memory (встроенное хранилище)

## Запуск локально

### Требования

- Gradle 7.5+ или Maven 3.5+
